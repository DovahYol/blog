### 问题一：
下面这条指令跳转到哪里？
```
   0x0000000000400f75 <+50>:	jmpq   *0x402470(,%rax,8)
```
这条指令以0x402470(,%rax,8)指向的内存的值作为地址，又有
```
(gdb) p $rax
$1 = 7
```
所以0x402470+7x8=0x4024a8，查询0x4024a8
```
(gdb) x 0x4024a8
0x4024a8:	0x00400fa6
```
得到跳转地址为0x00400fa6
